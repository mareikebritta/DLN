## Tables

You can make simple tables by using pipe symbol and minus signs:

| column 1 | column 2 | Column 3|
|---------:|:----------:|---------:|
| item 1 | item 2 | item 3|
|**bold** | *italics* | `code` |
| HUUUUUUUUUGE ITEM | item 2 | item 3|
| item 1 | item 2 | item 3|
| item 1 | item 2 | item 3|
